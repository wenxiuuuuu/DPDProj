### How to Run

1. Install Streamlit
   ```sh
   pip install streamlit
   ```
1. Install Plotly
   ```sh
   pip install plotly
   ```
3. Running the project
   Open the folder in your terminal and enter
   ```sh
   streamlit run main.py
   ```
   This should open a local host on your browser with the webapp running successfully.
